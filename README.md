# Map Viz
## Setup

```sh
npm install
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

Add the data file to the root folder with the name `data.geojson`

## Running

Open a terminal and type:

```sh
npm start
```

Open another terminal and type:

```sh
virtualenv venv
source venv/bin/activate
python app.py
```
